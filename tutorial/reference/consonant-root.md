## IDC 类型说明

| IDC | 符号   | 名称     | 组合方式    |

| ⿺  | U+2FFA | 左下包围 | 左下 + 右上 |

## 设计理念

声母字根的选择遵循以下原则：

1. **象形关联**：`t` → 七（t 像七的勾）、`h` → 丩（h 像丩的竖）
2. **语义关联**：`m` → 木（m 像木的声母）、`f` → 父（f 像父的声母）
3. **形状相似**：`n` → 冖（n 像冖的拱形）、`l` → 𠃊（l 像𠃊的竖折）
4. **独特性**：每个声母有唯一的字根，避免混淆
