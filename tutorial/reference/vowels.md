# 韵母编码表

37 个韵母按结构分为 7 类。音韵码 = 去除 `_` 占位符后的字符串。

## 韵头（3 个）

| 拼音 | 模板 | 音韵码 | 音韵字 | 部首 |
| ---- | ---- | ------ | ------ | ---- |
| `i`  | `i_` | `i`    | `⿱一` | 一   |
| `u`  | `u_` | `u`    | `⿱土` | 土   |
| `ü`  | `v_` | `v`    | `⿱女` | 女   |

## 韵腹（8 个）

| 拼音 | 模板   | 音韵码 | 音韵字 | 部首 |
| ---- | ------ | ------ | ------ | ---- |
| `a`  | `_aa_` | `aa`   | `⿱八` | 八   |
| `o`  | `_oo_` | `oo`   | `⿱乇` | 乇   |
| `e`  | `_ee_` | `ee`   | `⿱么` | 么   |
| `ai` | `_a_`  | `a`    | `⿱才` | 才   |
| `ei` | `_e_`  | `e`    | `⿱贝` | 贝   |
| `ao` | `_ao_` | `ao`   | `⿱刀` | 刀   |
| `ou` | `_o_`  | `o`    | `⿱斗` | 斗   |
| `er` | `_er_` | `er`   | `⿱耳` | 耳   |

> 注意：单韵母 `a`/`o`/`e` 在韵腹位置需要双写（`aa`/`oo`/`ee`），以与复韵母 `ai→a`、`ou→o`、`ei→e` 区分。

## 韵尾（2 个）

| 拼音  | 模板 | 音韵码 | 音韵字 | IDC 类型    |
| ----- | ---- | ------ | ------ | ----------- |
| `en`  | `_n` | `n`    | `⿵冂` | ⿵ 上包围   |
| `eng` | `_g` | `g`    | `⿹勹` | ⿹ 右上包围 |
