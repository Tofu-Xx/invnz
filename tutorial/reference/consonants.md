## 音韵码拼写规则总结

| 声母分组 | 规则                | 举例              |
| -------- | ------------------- | ----------------- |
| 普通     | 无特殊规则          | `b` + `a` → `baa` |
| jqx      | 后接 `u` 时写为 `v` | `j` + `u` → `jv`  |
| 可读     | 后接空韵 `i` 时省略 | `zh` + `i` → `zh` |
