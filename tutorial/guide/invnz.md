# 音韵字详解

音韵字（invnz）是**音韵码的汉字化表达**。每个音韵码组件（声母、韵头、韵腹、韵尾）对应一个汉字部首，然后用汉字结构指示符（IDC）组合成**看起来像一个汉字**的字符串。

## 核心思想

**音韵字与音韵码一一对应**，每段音韵码都有固定的音韵字表示：

| 音韵码段 | 含义     | 音韵字部件 |
| -------- | -------- | ---------- |
| `b`      | 声母 b   | `⿰匕`     |
| `zh`     | 声母 zh  | `⿰止`     |
| `i`      | 韵头 i   | `⿱一`     |
| `u`      | 韵头 u   | `⿱土`     |
| `a`      | 韵腹 ai  | `⿱才`     |
| `n`      | 韵尾 en  | `⿵冂`     |
| `g`      | 韵尾 eng | `⿹勹`     |

## 汉字结构指示符 (IDC)

使用 5 个 Unicode 汉字结构指示符来组合部件：

| IDC  | 名称     | 含义             |
| ---- | -------- | ---------------- |
| `⿰` | 左右结构 | 左部件 + 右部件  |
| `⿱` | 上下结构 | 上部件 + 下部件  |
| `⿸` | 左上包围 | 左上部件包围右下 |
| `⿹` | 右上包围 | 右上部件包围左下 |
| `⿵` | 上包围   | 上部部件向下包围 |

## 声母的音韵字

声母由 **IDC + 部首** 两部分组成：

| 声母 | 音韵字 | 结构     | 部首 |
| ---- | ------ | -------- | ---- |
| `b`  | `⿰匕` | 左右     | 匕   |
| `p`  | `⿰片` | 左右     | 片   |
| `m`  | `⿰木` | 左右     | 木   |
| `f`  | `⿰方` | 左右     | 方   |
| `d`  | `⿰丁` | 左右     | 丁   |
| `t`  | `⿰田` | 左右     | 田   |
| `n`  | `⿰牛` | 左右     | 牛   |
| `l`  | `⿰立` | 左右     | 立   |
| `g`  | `⿱工` | 上下     | 工   |
| `k`  | `⿰口` | 左右     | 口   |
| `h`  | `⿰火` | 左右     | 火   |
| `j`  | `⿱久` | 上下     | 久   |
| `q`  | `⿰七` | 左右     | 七   |
| `x`  | `⿰忄` | 左右     | 忄   |
| `zh` | `⿰止` | 左右     | 止   |
| `ch` | `⿸厂` | 左上包围 | 厂   |
| `sh` | `⿰山` | 左右     | 山   |
| `r`  | `⿰日` | 左右     | 日   |
| `z`  | `⿰子` | 左右     | 子   |
| `c`  | `⿱艹` | 上下     | 艹   |
| `s`  | `⿰纟` | 左右     | 纟   |

> 注意：`g`、`j`、`c` 是上下结构（`⿱`），`ch` 是左上包围（`⿸`），其余均为左右结构（`⿰`）。

## 韵母的音韵字

### 韵头 / 韵腹（独立）

| 音韵码段 | 拼音来源 | 音韵字 | 部首 |
| -------- | -------- | ------ | ---- |
| `i`      | i 韵头   | `⿱一` | 一   |
| `u`      | u 韵头   | `⿱土` | 土   |
| `v`      | ü 韵头   | `⿱女` | 女   |
| `a`      | ai       | `⿱才` | 才   |
| `e`      | ei       | `⿱贝` | 贝   |
| `o`      | ou       | `⿱斗` | 斗   |
| `aa`     | a        | `⿱八` | 八   |
| `oo`     | o        | `⿱乇` | 乇   |
| `ee`     | e        | `⿱么` | 么   |
| `ao`     | ao       | `⿱刀` | 刀   |
| `er`     | er       | `⿱耳` | 耳   |

### 韵尾

| 音韵码段 | 拼音来源 | 音韵字 | IDC 类型       |
| -------- | -------- | ------ | -------------- |
| `n`      | en       | `⿵冂` | 上包围（⿵）   |
| `g`      | eng      | `⿹勹` | 右上包围（⿹） |

韵尾的包裹方式与其他组件不同——它包围前一个组件（见下方组合规则）。

## 组合规则

### 规则 1：声母 + 韵腹（左右结构）

```
⿰声母部首 + 韵腹部首
```

| 音韵码     | 组合          | 音韵字   |
| ---------- | ------------- | -------- |
| `ba` (bai) | `⿰匕` + `才` | `⿰匕才` |
| `mao`      | `⿰木` + `刀` | `⿰木刀` |

### 规则 2：韵尾包裹前一个组件

韵尾（`⿵冂` / `⿹勹`）**包围**韵腹或声母：

| 音韵码      | 组合          | 音韵字   |
| ----------- | ------------- | -------- |
| `bn` (ben)  | `⿵冂` + `匕` | `⿵冂匕` |
| `bg` (beng) | `⿹勹` + `匕` | `⿹勹匕` |
| `an`        | `⿵冂` + `才` | `⿵冂才` |

### 规则 3：韵头包裹剩余部分

韵头（`⿱一`/`⿱土`/`⿱女`）在最外层用上下结构包裹：

| 音韵码       | 组合              | 音韵字       |
| ------------ | ----------------- | ------------ |
| `bia` (bie)  | `⿰匕` + `⿱一才` | `⿰匕⿱一才` |
| `lia` (lie)  | `⿰立` + `⿱一才` | `⿰立⿱一才` |
| `gua` (guai) | `⿱工` + `⿱土才` | `⿱工⿱土才` |

### 规则 4：韵头 + 韵尾包裹

当韵头（i/u/v）与韵尾（n/g）同时出现时，韵尾包裹韵腹，韵头再包裹整体：

| 音韵码       | 组合过程                      | 音韵字       |
| ------------ | ----------------------------- | ------------ |
| `bin`        | i → `⿱一`，n → `⿵冂`→包裹 a | `⿰匕⿵冂一` |
| `big` (bing) | i → `⿱一`，g → `⿹勹`→包裹 a | `⿰匕⿹勹一` |
| `dug` (dong) | u → `⿱土`，g → `⿹勹`→包裹 a | `⿰丁⿹勹土` |

### 规则 5：三组件（韵头 + 韵腹 + 韵尾）

韵尾包围韵腹，韵头再包裹这个组合：

| 音韵码         | 组合                  | 音韵字           |
| -------------- | --------------------- | ---------------- |
| `bian`         | `⿰匕` + `⿱一⿵冂才` | `⿰匕⿱一⿵冂才` |
| `guag` (guang) | `⿱工` + `⿱土⿹勹才` | `⿱工⿱土⿹勹才` |
| `zhuan`        | `⿰止` + `⿱土⿵冂才` | `⿰止⿱土⿵冂才` |

### 规则 6：可读声母空韵

zh/ch/sh/r/z/c/s 后无独立韵母时，音韵字只剩声母的**部首**：

| 音韵码     | 音韵字 |
| ---------- | ------ |
| `zh` (zhi) | `止`   |
| `ch` (chi) | `厂`   |
| `sh` (shi) | `山`   |
| `r` (ri)   | `日`   |
| `z` (zi)   | `子`   |
| `c` (ci)   | `艹`   |
| `s` (si)   | `纟`   |

### 规则 7：零声母特殊处理

零声母的音韵字直接从韵母模板生成，去掉外层 IDC：

| 音韵码   | 来源             | 音韵字 |
| -------- | ---------------- | ------ |
| `i` (yi) | `⿱一` → 去 `⿱` | `一`   |
| `u` (wu) | `⿱土` → 去 `⿱` | `土`   |
| `v` (yu) | `⿱女` → 去 `⿱` | `女`   |
| `aa` (a) | `⿱八` → 去 `⿱` | `八`   |
| `er`     | `⿱耳` → 去 `⿱` | `耳`   |

但复合零声母（如 `ia`、`ua`、`in` 等）仍然保留 IDC：

| 音韵码       | 来源     | 音韵字       |
| ------------ | -------- | ------------ |
| `ia` (ye)    | ie 模板  | `⿱一才`     |
| `uan` (wan)  | uan 模板 | `⿱土⿵冂才` |
| `van` (yuan) | üan 模板 | `⿱女⿵冂才` |
