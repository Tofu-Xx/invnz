# 音韵字详解

**音韵字（invnz）** 是 invnz 的终极输出——将音韵码的每个音素映射到汉字组件，再用汉字结构指示符（IDC）组合成一个"类汉字"字符串。

如果说音韵码是**乐谱**，那么音韵字就是**演奏出来的音乐**——看得见、摸得着。

## 从音素到字根

每个音素（phoneme）都有对应的**字根（wordroot）**——一个汉字部首或组件，可能带有 IDC 前缀。

## 组合规则

### 规则一：韵尾包围

当韵部中有韵尾（en / an / eng / ang）时，韵尾包裹前面的所有音素字根：

```
ie + an  →  ⿵𰃦也
      ↑        ↑
    韵尾    韵尾包也
```

```
i + en  →  ⿵冂一
```

```
u + ang  →  ⿹勹五
```

### 规则二：声母包裹

声母的 IDC 包裹整个韵部：

```
zh + ong（无韵尾，单一音素）
  ↓ 去 IDC → 工
  ↓ 声母包裹 → ⿱止 + 工 → ⿱止工
```

```
zh + en（仅有韵尾）
  ↓ 韵尾包裹？声母是 ⿰ 还是 ⿱？
```

当**仅有韵尾**（无其他韵母音素）时：

- 如果声母字根是 **`⿰`**（左右结构）：韵尾包裹声母的内核
  - `t` + `an` → t(⿰七) + an(⿵𰃦) → ⿵𰃦**七**
  - `h` + `an` → h(⿰丩) + an(⿵𰃦) → ⿵𰃦**丩**

- 如果声母字根是 **`⿱`**（上下结构）：声母包裹去 IDC 后的韵尾
  - `b` + `an` → b(⿱不) + an(⿵𰃦) → ⿱不**𰃦**
  - `d` + `en` → d(⿱𠂊) + en(⿵冂) → ⿱𠂊**冂**

- 其他（`⿺`、`⿸` 等）：声母包裹去 IDC 后的韵尾
  - `l` + `an` → l(⿺𠃊) + an(⿵𰃦) → ⿺𠃊**𰃦**
  - `sh` + `an` → sh(⿸尸) + an(⿵𰃦) → ⿸尸**𰃦**

### 规则三：⿰X一 → ⿱X一

这是一个特殊的修正规则：当组合结果恰好是 `⿰X一`（左右结构 + 一字在右）时，转为 `⿱X一`（上下结构 + 一字在下）。

```
⿰不一  →  ⿱不一  （bo）
⿰七一  →  ⿱七一  （ti）
```

> 这条规则让更多结果看起来像"上下结构"的自然汉字。

### 规则四：零声母处理

零声母音节（没有声母）在匹配 `-x` 前缀条目时，加上 `⿰丨` 前缀：

```
a    → 匹配 -a  →  ⿰丨八
wo   → 匹配 -wo →  ⿰丨火
ong  → 匹配 -ong →  ⿰丨工
```

其他零声母音节通过流水线处理，不加 `⿰丨`。

## 组合流程示例

### 示例：zhong（zh + ong）

```
1. 音韵码：zh_ong
2. 查字根：
   zh   → ⿱止
   ong  → 工
3. 韵部：[工]（单一音素，非韵尾）
   去 IDC → 工
4. 声母包裹：
   ⿱止 + 工 → ⿱止工
```

### 示例：tian（t + ie + an）

```
1. 音韵码：tie_an
2. 查字根：
   t    → ⿰七
   ie   → 也
   an   → ⿵𰃦
3. 韵部：[也, ⿵𰃦]
   韵尾⿵𰃦包裹也 → ⿵𰃦也
4. 声母⿰七包裹 ⿵𰃦也 → ⿰七⿵𰃦也
```

### 示例：ban（b + an）

```
1. 音韵码：b_an
2. 查字根：
   b    → ⿱不
   an   → ⿵𰃦
3. 韵部：[⿵𰃦]（仅有韵尾）
   声母是 ⿱（非 ⿰）
   → 声母包裹去 IDC 后的韵尾
   → ⿱不 + 𰃦 → ⿱不𰃦
```

### 示例：han（h + an）

```
1. 音韵码：h_an
2. 查字根：
   h    → ⿰丩
   an   → ⿵𰃦
3. 韵部：[⿵𰃦]（仅有韵尾）
   声母是 ⿰
   → 韵尾包裹声母内核
   → ⿵𰃦 + 丩 → ⿵𰃦丩
```

### 示例：wu（整体）

```
wu 映射到「五」字，不走流水线。
```

### 示例：yong（整体）

```
yong 映射到「用」字，不走流水线。
```
