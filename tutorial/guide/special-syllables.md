# 特殊音节

大部分音节通过"拆解音素 → 映射字根 → 组合成形"的流水线处理。但有一些音节采用了特殊处理。

## 零声母音节

零声母音节（以 `y`、`w` 开头或以元音开头）没有声母。它们的处理分为两种情况：

### 带 `⿰丨` 的零声母

以下零声母音节在流水线中加 `⿰丨`（丨在左，表示零声母）：

| 拼音 | 音韵字 | 说明        |
| ---- | ------ | ----------- |
| a    | ⿰丨八 | 零声母 + 八 |
| o    | ⿰丨口 | 零声母 + 口 |
| e    | ⿰丨戈 | 零声母 + 戈 |
| ai   | ⿰丨才 | 零声母 + 才 |
| ei   | ⿰丨巿 | 零声母 + 巿 |
| ao   | ⿰丨刀 | 零声母 + 刀 |
| ou   | ⿰丨口 | 零声母 + 口 |
| wo   | ⿰丨火 | 零声母 + 火 |
| ong  | ⿰丨工 | 零声母 + 工 |

### 不带 `⿰丨` 的零声母

以下零声母音节通过韵母分解流水线处理，**不**加 `⿰丨`：

| 拼音 | 音韵字 | 分解                   |
| ---- | ------ | ---------------------- |
| ya   | ⿱一八 | i_a（介音 i + 韵腹 a） |
| yao  | ⿱一刀 | i_ao                   |
| you  | ⿱一口 | i_ou                   |
| yan  | ⿵𰃦也 | ie_an                  |
| yang | ⿹勹一 | i_ang                  |
| wa   | ⿱五八 | u_a                    |
| wai  | ⿱五才 | u_ai                   |
| wei  | ⿱五巿 | u_ei                   |
| wan  | ⿵𰃦五 | u_an                   |
| wen  | ⿵冂五 | u_en                   |
| wang | ⿹勹五 | u_ang                  |

## 直接映射音节

以下音节不经过音素拆解，**整体直接映射**为一个字符：

| 拼音 | 字根 | 说明     |
| ---- | ---- | -------- |
| yi   | 一   | 整体象形 |
| wu   | 五   | 整体象形 |
| yu   | 雨   | 整体象形 |
| ye   | 也   | 整体象形 |
| yue  | 月   | 整体象形 |
| yong | 用   | 整体象形 |
| er   | 耳   | 整体象形 |

## 无声母包裹的特殊音节

以下音节虽然有声母，但映射结果直接等于**韵母字根**（声母不包裹）：

| 拼音 | 音韵字 | 说明           |
| ---- | ------ | -------------- |
| ge   | 戈     | 直接映射为字根 |
| gong | 工     | 直接映射为字根 |
| dao  | 刀     | 直接映射为字根 |
| kou  | 口     | 直接映射为字根 |
| huo  | 火     | 直接映射为字根 |
| pei  | 巿     | 直接映射为字根 |
| cai  | 才     | 直接映射为字根 |

这些音节实际上在流水线之前就被 `getDirectSyllable` 直接匹配，返回韵母字根本身。

## 独立韵尾音节

以下音节只有韵尾，没有其他韵母音素，直接映射为韵尾字根（不带 `⿰丨`）：

| 拼音 | 字根 |
| ---- | ---- |
| an   | 𰃦   |
| en   | 冂   |
| ang  | 勹   |
| eng  | 𠃌   |
