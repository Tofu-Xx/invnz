## IDC 类型

汉字结构指示符（Ideographic Description Character）用于描述汉字的结构：

| IDC | 符号 | 名称 | 含义 |
|-----|------|------|------|
| ⿰ | U+2FF0 | 左右结构 | 左 + 右 |
| ⿱ | U+2FF1 | 上下结构 | 上 → 下 |
| ⿵ | U+2FF5 | 上三包围 | 上 → 下 |
| ⿸ | U+2FF8 | 左上包围 | 左上 → 右下 |
| ⿹ | U+2FF9 | 右上包围 | 右上 → 左下 |
| ⿺ | U+2FFA | 左下包围 | 左下 → 右上 |

## 设计理念

字根的选择遵循以下原则：

- **象形关联**：`t` → 七（t 的形状像七的勾）、`h` → 丩（h 像丩的竖）
- **语义关联**：`m` → 木（m 像木的声母）、`f` → 父（f 像父的声母）
- **形状相似**：`n` → 冖（n 像冖的拱形）、`l` → 𠃊（l 像𠃊的竖折）
- **独特性**：每个音素都有唯一的字根，避免混淆
