# 音韵码详解

音韵码（invn）是一种**双字母注音拼音方案**，每个音节用固定模式的短编码表示。

## 声母段

声母段与标准拼音**完全一致**，分为三组：

1. **普通声母**（11 个）：`b` `p` `m` `f` `d` `t` `n` `l` `g` `k` `h`
2. **jqx 声母**（3 个）：`j` `q` `x`（触发 u → v 转换）
3. **可读声母**（7 个）：`zh` `ch` `sh` `r` `z` `c` `s`（触发空韵 i 省略）

## 韵母段

韵母段来自韵母模板中**去除所有 `_` 占位符**后的字符串。韵母按结构分为 7 类：

### 韵头（3 个）

| 拼音 | 模板 | 音韵码 |
| ---- | ---- | ------ |
| `i`  | `i_` | `i`    |
| `u`  | `u_` | `u`    |
| `ü`  | `v_` | `v`    |

### 韵腹（8 个）

| 拼音 | 模板   | 音韵码 |
| ---- | ------ | ------ |
| `a`  | `_aa_` | `aa`   |
| `o`  | `_oo_` | `oo`   |
| `e`  | `_ee_` | `ee`   |
| `ai` | `_a_`  | `a`    |
| `ei` | `_e_`  | `e`    |
| `ao` | `_ao_` | `ao`   |
| `ou` | `_o_`  | `o`    |
| `er` | `_er_` | `er`   |

> 注意：`a`、`o`、`e` 在韵腹位置需要双写（`aa` / `oo` / `ee`），以与 `ai→a`、`ou→o`、`ei→e` 区分。

### 韵尾（2 个）

| 拼音  | 模板 | 音韵码 |
| ----- | ---- | ------ |
| `en`  | `_n` | `n`    |
| `eng` | `_g` | `g`    |

韵尾的 `n` / `g` 在 invnz 中会包裹前一个组件（见[音韵字详解](/guide/invnz)）。

### 头腹（9 个）

| 拼音  | 模板    | 音韵码 |
| ----- | ------- | ------ |
| `ie`  | `i_a_`  | `ia`   |
| `iu`  | `i_o_`  | `io`   |
| `ia`  | `i_aa_` | `iaa`  |
| `iao` | `i_ao_` | `iao`  |
| `uai` | `u_a_`  | `ua`   |
| `ui`  | `u_e_`  | `ue`   |
| `ua`  | `u_aa_` | `uaa`  |
| `uo`  | `u_oo_` | `uoo`  |
| `üe`  | `v_a_`  | `va`   |

### 头尾（5 个）

| 拼音  | 模板  | 音韵码 |
| ----- | ----- | ------ |
| `in`  | `i_n` | `in`   |
| `ing` | `i_g` | `ig`   |
| `un`  | `u_n` | `un`   |
| `ong` | `u_g` | `ug`   |
| `ün`  | `v_n` | `vn`   |

### 腹尾（2 个）

| 拼音  | 模板   | 音韵码 |
| ----- | ------ | ------ |
| `an`  | `_a_n` | `an`   |
| `ang` | `_a_g` | `ag`   |

### 头腹尾（6 个）

| 拼音   | 模板    | 音韵码 |
| ------ | ------- | ------ |
| `ian`  | `i_a_n` | `ian`  |
| `iang` | `i_a_g` | `iag`  |
| `iong` | `i_u_g` | `iug`  |
| `uan`  | `u_a_n` | `uan`  |
| `uang` | `u_a_g` | `uag`  |
| `üan`  | `v_a_n` | `van`  |

## 完整音节示例

| 拼音     | 声母 | 韵母模板 | 音韵码  |
| -------- | ---- | -------- | ------- |
| `zhuang` | `zh` | `u_a_g`  | `zhuag` |
| `liang`  | `l`  | `i_a_g`  | `liag`  |
| `jiong`  | `j`  | `i_u_g`  | `jiug`  |
| `feng`   | `f`  | `_g`     | `fg`    |
| `tian`   | `t`  | `i_a_n`  | `tian`  |
| `huan`   | `h`  | `u_a_n`  | `huan`  |
