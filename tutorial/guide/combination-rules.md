# 组合规则详解

有了字根之后，如何把它们组合成一个完整的音韵字？本文详细解释四条组合规则。

## 规则一：韵尾包围

四个韵尾（en、an、eng、ang）拥有包裹能力——它们用自身 IDC **包住**前面的字根。

```text
ie + an
  ie → 也
  an → ⿵𰃦
  ⿵𰃦 + 也 → ⿵𰃦也
```

```text
i + en
  i → 一
  en → ⿵冂
  ⿵冂 + 一 → ⿵冂一
```

```text
u + ang
  u → 五
  ang → ⿹勹
  ⿹勹 + 五 → ⿹勹五
```

## 规则二：声母包裹

声母字根包裹整个韵部（韵母组合的结果）：

```text
例：zhong（zh + ong）

韵部：ong → 工（单一音素，去 IDC → 工）
声母：zh → ⿱止
组合：⿱止 + 工 → ⿱止工
```

```text
例：tian（t + ie + an）

韵部：ie + an → ⿵𰃦也（韵尾包裹 ie 的字根）
声母：t → ⿰七
组合：⿰七 + ⿵𰃦也 → ⿰七⿵𰃦也
```

```text
例：guan（g + u + an）

韵部：u + an → ⿵𰃦五（韵尾 an 包裹 u）
声母：g → ⿰𠂎
组合：⿰𠂎 + ⿵𰃦五 → ⿰𠂎⿵𰃦五
```

## 规则三：仅有韵尾时的特殊处理

当韵部**只有一个韵尾**（没有其他韵母音素）时，规则二的包裹方向取决于声母字根的 IDC 类型：

### 声母 IDC = ⿰（左右结构）→ 韵尾包裹声母内核

```text
例：tan（t + an）

声母 t → ⿰七（⿰ 结构）
韵尾 an → ⿵𰃦
组合：⿵𰃦 + 七（内核）→ ⿵𰃦七
```

```text
例：han（h + an）

声母 h → ⿰丩（⿰ 结构）
韵尾 an → ⿵𰃦
组合：⿵𰃦 + 丩 → ⿵𰃦丩
```

```text
例：gen（g + en）

声母 g → ⿰𠂎（⿰ 结构）
韵尾 en → ⿵冂
组合：⿵冂 + 𠂎 → ⿵冂𠂎
```

### 声母 IDC = ⿱ 或其他 → 声母包裹韵尾

```text
例：ban（b + an）

声母 b → ⿱不（⿱ 结构）
韵尾 an → ⿵𰃦，去 IDC → 𰃦
组合：⿱不 + 𰃦 → ⿱不𰃦
```

```text
例：den（d + en）

声母 d → ⿱𠂊（⿱ 结构）
韵尾 en → ⿵冂，去 IDC → 冂
组合：⿱𠂊 + 冂 → ⿱𠂊冂
```

```text
例：lan（l + an）

声母 l → ⿺𠃊（⿺ 包围结构）
韵尾 an → ⿵𰃦，去 IDC → 𰃦
组合：⿺𠃊 + 𰃦 → ⿺𠃊𰃦
```

```text
例：shan（sh + an）

声母 sh → ⿸尸（⿸ 包围结构）
韵尾 an → ⿵𰃦，去 IDC → 𰃦
组合：⿸尸 + 𰃦 → ⿸尸𰃦
```

### 判定规则总结

| 声母 IDC     | 包裹方向   | 示例          |
| ------------ | ---------- | ------------- |
| ⿰（左右）   | 韵尾包声母 | tan → ⿵𰃦七  |
| ⿱（上下）   | 声母包韵尾 | ban → ⿱不𰃦  |
| ⿸（左上包） | 声母包韵尾 | shan → ⿸尸𰃦 |
| ⿺（左下包） | 声母包韵尾 | lan → ⿺𠃊𰃦  |

## 规则四：⿰X一 → ⿱X一

当组合结果恰好是 `⿰X一`（左右结构，右边是「一」）时，自动修正为 `⿱X一`（上下结构，下面是「一」）：

```text
⿰不一 → ⿱不一（bu 的音韵字）
⿰七一 → ⿱七一（ti 的音韵字）
```

这条规则让更多结果看起来像常见的上下结构汉字。

## 全流程示例

### zhong

```
1. 拆解：zh + ong
2. 查字根：⿱止 + 工
3. 韵部：[工]（单一非韵尾，去 IDC → 工）
4. 声母包韵部：⿱止 + 工 → ⿱止工
```

### tian

```
1. 拆解：t + ie + an
2. 查字根：⿰七 + 也 + ⿵𰃦
3. 韵部：[也, ⿵𰃦] → ⿵𰃦也（韵尾包 ie）
4. 声母包韵部：⿰七 + ⿵𰃦也 → ⿰七⿵𰃦也
```

### shuang

```
1. 拆解：sh + u + ang
2. 查字根：⿸尸 + 五 + ⿹勹
3. 韵部：[五, ⿹勹] → ⿹勹五（韵尾包 u）
4. 声母包韵部：⿸尸 + ⿹勹五 → ⿸尸⿹勹五
```

### han

```
1. 拆解：h + an
2. 查字根：⿰丩 + ⿵𰃦
3. 韵部：[⿵𰃦]（仅有韵尾）
4. 声母是 ⿰ → 韵尾包内核
5. ⿵𰃦 + 丩 → ⿵𰃦丩
```
