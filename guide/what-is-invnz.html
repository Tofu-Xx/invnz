invnz 是什么 ​

invnz 是一个将汉语拼音转换为「音韵字」的工具——把拼音的每一个音素映射为一个汉字部件，再用汉字的结构规则组合起来。

结果是一个看起来像汉字、实际上表达拼音的字符。

为什么需要它？ ​

拼音用拉丁字母记录汉语发音。但字母是线性的——zhong 只是 5 个字母的排列，我们看不到声母在哪里结束、韵母从哪里开始。

汉字则不同。汉字是结构化的——上下、左右、包围，每一种结构都传递着信息。

invnz 把拼音的结构可视化出来：

zhong（5 个字母，线性排列）
  ↓
⿱止工（3 个部件，结构清晰）
 ├─ 止（声母 zh）
 └─ 工（韵母 ong）

它是怎么工作的？ ​

invnz 的工作方式可以概括为三步：

1. 拆解音素 ​

把拼音拆成最小的发音单位——音素（phoneme）。

zhong → zh + ong
tian  → t + ie + an
shuang → sh + u + ang

2. 映射字根 ​

每个音素对应一个汉字部件（称为字根）：

音素字根说明
zh止声母 zh 的象形
t七声母 t 的象形
ie也韵母 ie 的字根
an𰃦韵尾 an 的包围形
ong工韵母 ong 的字根

3. 组合成形 ​

用汉字结构指示符（IDC）将字根组合起来：

止（上下结构） → ⿱止
工              → 工
⿱止 + 工       → ⿱止工

它不是什么？ ​

  • 不是新的拼音方案 — 它不改变读音，只是把读音可视化
  • 不是一个字体 — 它输出的是 IDS 字符串，不是渲染好的字形
  • 不是翻译工具 — 它不翻译汉字，只编码拼音

它能做什么？ ​

  • 教育：帮助学习者理解拼音的声韵结构
  • 设计：作为一种视觉元素，把拼音变成图形
  • 研究：探索汉语音韵与汉字结构的对应关系

继续阅读内部原理，深入了解音素拆解和字根组合的完整过程。
