## invenma规则

### 1.1 关于v

代表ü，包括jqxy后面的u(实际上也是ü)

### 1.2 关于i

j q x zh ch sh r z c s 后面省去韵母i

因为这些声母的本身读法，和后面加上i表示的读音一致，故省去
整体part的除外（如ie,iong），因为它们有对应字根

### 1.3 关于o

b p m f后面的韵母o变成uo

## invenz规则

### 2.1 关于'一'

当左右结构字，右侧只有'一'时，整体变成上下结构

### 2.2 关于'⿱冂','⿱𰃦','⿹𠃌','⿹勹'

包含前面的一个字根，也就是和前面一个字符调换位置

### 2.3 关于'丨'

用于部分的零声母音节('-xx')前面，用于提示
