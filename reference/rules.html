特殊转换 ​

i‑drop：当声母为 j q x zh ch sh r z c s 时，韵母中独立的 i 音素被省略：

ji     → j_i   （不变 — 无后续韵母）
jin    → j_en  （i 被省略）
jing   → j_eng （i 被省略）
xin    → x_en  （i 被省略）

bpmf + o → uo：

bo  → b_uo
po  → p_uo
mo  → m_uo
fo  → f_uo

jqx + u → ü：

ju   → j_v
qu   → q_v
xu   → x_v
juan → j_ve_an

四、IDC 组合规则 ​

规则 1：韵尾包围 ​

韵尾（en/an/eng/ang）的 IDC 包裹前一个字根：

i + en     → ⿵冂一
ie + an    → ⿵𰃦也
u + eng    → ⿹𠃌五
i + ang    → ⿹勹一

规则 2：声母包裹 ​

声母的 IDC 包裹整个韵部组合结果：

zh + ong       → ⿱止工
g + u + an     → ⿰𠂎⿵𰃦五
t + ie + an    → ⿰七⿵𰃦也

规则 3：仅有韵尾 ​

当韵部只有一个韵尾时：

  • 声母 IDC = ⿰：韵尾包裹声母的内核

    • t + an → ⿵𰃦七
    • h + an → ⿵𰃦丩

  • 声母 IDC = ⿱ 或其他：声母包裹去 IDC 后的韵尾

    • b + an → ⿱不𰃦
    • d + en → ⿱𠂊冂
    • l + an → ⿺𠃊𰃦
    • sh + an → ⿸尸𰃦

规则 4：⿰X一 → ⿱X一 ​

当组合结果正好为 ⿰X一（3 个 Unicode 字符）时，转为 ⿱X一：

⿰不一 → ⿱不一
⿰七一 → ⿱七一

规则 5：零声母 ​

仅当音节通过 -x 前缀条目匹配时，加 ⿰丨：

a   → -a 匹配 → ⿰丨八
wo  → -wo 匹配 → ⿰丨火
ong → -ong 匹配 → ⿰丨工

五、特殊情况 ​

音节处理方式
wu直接映射为「五」
yi直接映射为「一」
yu直接映射为「雨」
ye直接映射为「也」
yue直接映射为「月」
yong直接映射为「用」
er直接映射为「耳」
an/en/ang/eng直接映射为韵尾字根（无 ⿰丨）
